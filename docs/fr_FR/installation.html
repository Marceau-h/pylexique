Installation¶

Version stable¶

Pour installer pylexique, exécutez cette commande dans votre terminal :

$ pip install pylexique

C’est la méthode recommandée pour installer pylexique, car elle installera toujours la version stable la plus récente.

Si vous n’avez pas installé pip, ce guide d’installation de Python peut vous guider dans le processus.

Vous pouvez également installer pylexique en utilisant Anaconda ou Miniconda au lieu de pip .

Pour installer Anaconda ou Miniconda, veuillez suivre les instructions d’installation sur leurs sites web respectifs. Après avoir installé Anaconda ou Miniconda, exécutez ces commandes dans votre terminal :

$ conda config --add channels conda-forge
$ conda config --set channel_priority strict
$ conda install pylexique

Si vous avez déjà Anaconda ou Miniconda disponible sur votre système, tapez simplement ceci dans votre terminal :

$ conda install -c conda-forge pylexique

Avertissement

Si vous avez l’intention d’installer pylexique sur un Apple Macbook avec un processeur Apple M1, il est conseillé d’installer pylexique en utilisant la méthode d’installation conda car toutes les dépendances seront précompilées.

Depuis le code source¶

Le code source de pylexique peut être téléchargé depuis le dépôt Github.

Vous pouvez soit cloner le dépôt public :

$ git clone git://github.com/SekouDiaoNlp/pylexique

Ou téléchargez le fichier tarball :

$ curl  -OL https://github.com/SekouDiaoNlp/pylexique/tarball/master

Une fois que vous avez une copie du code source, vous pouvez l’installer avec :

$ python setup.py install
